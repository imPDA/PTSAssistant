# PTS Assistant
An ESO addon to locate items within your bags and automatically retrieve them through nested containers.

### Features
- **Locate Items**: Instantly shows which bag container holds a specific item.
- **Auto-Retrieve**: Automatically opens all containers in the chain to retrieve the target item.
- **Searchable Interface**: Filter your items quickly using the search box.

### How to Use
- Open inventory.
- Type in the search box to filter items.
- Double-click (or right-click) on an item to automatically retrieve it.
